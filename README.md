# うちなリアン 琉 — 公式サイト

大和市桜ヶ丘の創作沖縄料理店「うちなリアン 琉」の公式ウェブサイトです。

## サイト構成

```
uchinalian-ryu/
├── index.html      # メインページ
├── style.css       # スタイルシート
├── main.js         # JavaScript
├── README.md       # このファイル
└── images/         # 画像ファイル
    ├── exterior.jpeg    # 外観
    ├── interior1.jpeg   # 店内（テーブル席）
    ├── interior2.jpeg   # 店内（カウンター）
    ├── awamori.jpeg     # 泡盛コレクション
    ├── tacorice.jpeg    # 炙りチーズタコライス
    ├── soki.jpeg        # 豚出汁ソーキそば
    ├── rafute.jpeg      # 脂身とろとろラフテー
    ├── goya.jpeg        # 沖縄県産ゴーヤ
    ├── kakuni.jpeg      # 豚バラ肉（仕込み）
    └── umibudo.jpeg     # 海ぶどう
```

## GitHub Pages での公開方法

1. このリポジトリを GitHub にプッシュ
2. **Settings** → **Pages** を開く
3. **Source** で `main` ブランチ / `/ (root)` を選択して **Save**
4. 数分後に `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開されます
